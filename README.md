## Building/Running

```bash
$ git clone https://github.com/wchang22/RTRayTracer.git
$ mkdir -p RTRayTracer/build/release && cd RTRayTracer/build/release
$ cmake ../..
$ make all
$ ./rtraytracer
```

## Controls

* Forward, Left, Back, Right: `WASD`
* Up, Down: `space`, `x`
* Rotate view: `mouse`
